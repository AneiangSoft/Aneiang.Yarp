# 灵活端口 + 灵活启动 + 健康检查 + 配置热更新 方案设计

> 版本: v2.0  
> 日期: 2026-06-17  
> 状态: 设计完成，待实施

---

## 一、目标

1. **端口灵活配置**：支持任意端口、任意绑定（IP/HTTPS/IPv6）
2. **启动方式灵活配置**：
   - 兼容现有 AllInOne 模式（零配置升级）
   - 支持单进程多端口拆分模式
   - 支持独立 Proxy / Dashboard 进程
3. **健康检查**：提供 `/health` `/ready` `/live` 端点供 K8s/负载均衡探活
4. **配置热更新**：监听 `appsettings.json` 变化，自动重载业务配置
5. **安全防护**：Dashboard/Admin 端口误绑公网时给出告警，支持强制 Loopback 校验

---

## 三、启动模式枚举

| 模式 | 行为 | 适用场景 |
|------|------|---------|
| `Auto`（默认） | 根据 Endpoint 数量自动判断 | 智能适配 |
| `AllInOne` | 强制单端口，Dashboard + Proxy 同端口 | 兼容现有用户 |
| `Split` | 强制多端口，按角色路由 | 单机多端口隔离 |
| `ProxyOnly` | 禁用 Dashboard UI/API | 独立 Proxy 进程 |
| `DashboardOnly` | 禁用 YARP 转发 | 独立 Dashboard 进程 |

**自动检测规则**：
- 1 个 Endpoint → `AllInOne`
- 2+ 个 Endpoint → `Split`

```json
  }
```

```json
        "CheckDatabase":    true,
        "CheckConfigLoaded": true
      },
      "HotReload": {
        "Enabled":              true,
        "DebounceMilliseconds": 500,
        "WatchedFiles": [
          "appsettings.json",
          "appsettings.{Environment}.json"
        ],
        "FallbackPollSeconds": 30,
        "RollbackOnFailure":  true,
        "MaxSnapshots":       5
      }
    }
  }
}
```

### 4.3 角色枚举值

| Role | 允许的路径前缀 | 默认鉴权 |
|------|---------------|----------|
| `Proxy` | 所有非 Dashboard 路径 | 无（代理） |
| `Dashboard` | `/apigateway/*`, `/_content/*`, `/hubs/*`, `/api/*` | JWT/ApiKey |
| `Admin` | `/api/admin/*` | 本机 |
| `Health` | `/health`, `/ready`, `/live` | 无（可配置 Token） |
| `All` | 所有路径（AllInOne 默认） | 视情况 |

### 4.4 配置源优先级

```
从高到低：
1. 命令行参数（--Gateway:Deployment:Mode=Split）
2. 环境变量（Gateway__Deployment__Mode=Split）
3. appsettings.{ASPNETCORE_ENVIRONMENT}.json
4. appsettings.json
```

`IConfigurationRoot.Reload()` 会重新读取 3 和 4（不重新读取 1 和 2）。

### 5.2 EndpointRoleResolver

```csharp
public class EndpointRoleResolver
{
    private readonly Dictionary<int, EndpointRoleMap> _byPort = new();

    public EndpointRoleResolver(IConfiguration config, DeploymentOptions options)
    {
        var kestrelSection = config.GetSection("Kestrel:Endpoints");
        var endpointCount = kestrelSection.Exists() ? kestrelSection.GetChildren().Count() : 1;
        var resolvedMode = ResolveMode(options.Mode, endpointCount);

        foreach (var endpoint in kestrelSection.GetChildren())
        {
            var url = endpoint["Url"];
            if (string.IsNullOrEmpty(url)) continue;
            if (!Uri.TryCreate(url, UriKind.Absolute, out var uri)) continue;

            var role = ResolveRole(endpoint.Key, options.EndpointRoles, resolvedMode);
            _byPort[uri.Port] = new EndpointRoleMap
            {
                EndpointName = endpoint.Key,
                Port = uri.Port,
                IpAddress = uri.Host,
                Role = role,
                IsPubliclyBound = IsPublicBind(uri.Host)
            };
            options.ResolvedEndpoints.Add(_byPort[uri.Port]);
        }

        options.Mode = resolvedMode;
    }

    public EndpointRoleMap? GetByPort(int port) =>
        _byPort.TryGetValue(port, out var map) ? map : null;

    public IEnumerable<EndpointRoleMap> GetAll() => _byPort.Values;

    private static DeploymentMode ResolveMode(DeploymentMode configured, int count) =>
        configured != DeploymentMode.Auto
            ? configured
            : (count > 1 ? DeploymentMode.Split : DeploymentMode.AllInOne);

    private static string ResolveRole(string name, Dictionary<string, string> roles, DeploymentMode mode)
    {
        if (roles.TryGetValue(name, out var r)) return r;
        var lower = name.ToLowerInvariant();
        if (lower.Contains("proxy"))     return "Proxy";
        if (lower.Contains("dashboard")) return "Dashboard";
        if (lower.Contains("admin"))     return "Admin";
        if (lower.Contains("health"))    return "Health";
        return mode == DeploymentMode.AllInOne ? "All" : "Proxy";
    }

    private static bool IsPublicBind(string host) =>
        host == "0.0.0.0" || host == "*" || host == "::" || string.IsNullOrEmpty(host);
}
```

### 5.3 EndpointRouterMiddleware

```csharp
public class EndpointRouterMiddleware
{
    private readonly RequestDelegate _next;
    private readonly EndpointRoleResolver _resolver;
    private readonly DeploymentOptions _options;
    private readonly ILogger<EndpointRouterMiddleware> _logger;
    private readonly string _dashPrefix;

    private const string ContentRoot = "/_content/Aneiang.Yarp.Dashboard";
    private const string SignalRPrefix = "/hubs/";

    public EndpointRouterMiddleware(
        RequestDelegate next,
        EndpointRoleResolver resolver,
        IOptions<DeploymentOptions> options,
        ILogger<EndpointRouterMiddleware> logger,
        IOptions<DashboardOptions> dashOptions)
    {
        _next = next;
        _resolver = resolver;
        _options = options.Value;
        _logger = logger;
        _dashPrefix = "/" + dashOptions.Value.RoutePrefix.Trim('/');
    }

    public async Task InvokeAsync(HttpContext context)
    {
        if (_options.Mode == DeploymentMode.AllInOne)
        {
            await _next(context);
            return;
        }

        var path = context.Request.Path.Value ?? "/";

        if (_options.Mode == DeploymentMode.ProxyOnly)
        {
            if (IsDashboardPath(path)) { await Reject(context); return; }
            await _next(context);
            return;
        }

        if (_options.Mode == DeploymentMode.DashboardOnly)
        {
            if (!IsDashboardPath(path)) { await Reject(context); return; }
            await _next(context);
            return;
        }

        var localPort = context.Connection.LocalPort;
        var roleMap = _resolver.GetByPort(localPort);
        if (roleMap == null) { await Reject(context); return; }

        bool allowed = roleMap.Role switch
        {
            "All"       => true,
            "Health"    => IsHealthPath(path),
            "Admin"     => path.StartsWith("/api/admin", StringComparison.OrdinalIgnoreCase),
            "Dashboard" => IsDashboardPath(path),
            "Proxy"     => !IsDashboardPath(path),
            _           => true
        };

        if (!allowed)
        {
            _logger.LogDebug("Rejecting {Path} on port {Port} (role={Role})", path, localPort, roleMap.Role);
            await Reject(context);
            return;
        }

        await _next(context);
    }

    private bool IsDashboardPath(string path) =>
        path.StartsWith(_dashPrefix, StringComparison.OrdinalIgnoreCase) ||
        path.StartsWith(ContentRoot, StringComparison.OrdinalIgnoreCase) ||
        path.StartsWith(SignalRPrefix, StringComparison.OrdinalIgnoreCase) ||
        path.StartsWith("/api/", StringComparison.OrdinalIgnoreCase);

    private bool IsHealthPath(string path) =>
        path.Equals(_options.HealthCheck.Path, StringComparison.OrdinalIgnoreCase) ||
        path.Equals(_options.HealthCheck.ReadyPath, StringComparison.OrdinalIgnoreCase) ||
        path.Equals(_options.HealthCheck.LivePath, StringComparison.OrdinalIgnoreCase);

    private static async Task Reject(HttpContext context)
    {
        context.Response.StatusCode = 404;
        context.Response.ContentType = "application/json";
        await context.Response.WriteAsJsonAsync(new
        {
            error = "Not Found",
            message = "The requested resource is not available on this endpoint."
        });
    }
}
```

### 5.4 HealthCheckMiddleware

```csharp
public class HealthCheckMiddleware
{
    private readonly RequestDelegate _next;
    private readonly DeploymentOptions _options;
    private readonly IDynamicYarpConfigService _configService;
    private readonly IGatewayRepository? _repository;

    public HealthCheckMiddleware(
        RequestDelegate next,
        IOptions<DeploymentOptions> options,
        IDynamicYarpConfigService configService,
        IGatewayRepository? repository = null)
    {
        _next = next;
        _options = options.Value;
        _configService = configService;
        _repository = repository;
    }

    public async Task InvokeAsync(HttpContext context)
    {
        if (!_options.HealthCheck.Enabled) { await _next(context); return; }

        var path = context.Request.Path.Value ?? "/";
        if (!IsHealthPath(path)) { await _next(context); return; }

        if (_options.HealthCheck.AllowedIps.Any())
        {
            var clientIp = context.Connection.RemoteIpAddress?.ToString() ?? "";
            if (!_options.HealthCheck.AllowedIps.Contains(clientIp))
            {
                context.Response.StatusCode = 403;
                return;
            }
        }

        if (!string.IsNullOrEmpty(_options.HealthCheck.Token))
        {
            var token = context.Request.Headers["X-Health-Token"].FirstOrDefault()
                      ?? context.Request.Query["token"].FirstOrDefault();
            if (token != _options.HealthCheck.Token)
            {
                context.Response.StatusCode = 401;
                return;
            }
        }

        if (path.Equals(_options.HealthCheck.LivePath, StringComparison.OrdinalIgnoreCase))
        {
            await WriteJson(context, 200, new
            {
                status = "alive",
                timestamp = DateTime.UtcNow,
                uptime = (DateTime.UtcNow - Process.GetCurrentProcess().StartTime.ToUniversalTime()).TotalSeconds
            });
            return;
        }

        var (isReady, checks) = await PerformReadinessCheck(context.RequestAborted);

        if (path.Equals(_options.HealthCheck.ReadyPath, StringComparison.OrdinalIgnoreCase))
        {
            await WriteJson(context, isReady ? 200 : 503, new
            {
                status = isReady ? "ready" : "not-ready",
                checks
            });
            return;
        }

        if (path.Equals(_options.HealthCheck.Path, StringComparison.OrdinalIgnoreCase))
        {
            await WriteJson(context, isReady ? 200 : 503, new
            {
                status = isReady ? "healthy" : "degraded",
                timestamp = DateTime.UtcNow,
                version = Assembly.GetExecutingAssembly().GetName().Version?.ToString(),
                endpoints = _options.ResolvedEndpoints.Select(e => new
                {
                    name = e.EndpointName,
                    port = e.Port,
                    role = e.Role,
                    public = e.IsPubliclyBound
                }),
                uptime = (DateTime.UtcNow - Process.GetCurrentProcess().StartTime.ToUniversalTime()).TotalSeconds,
                checks
            });
        }
    }

    private async Task<(bool, Dictionary<string, object>)> PerformReadinessCheck(CancellationToken ct)
    {
        var checks = new Dictionary<string, object>();
        var isReady = true;

        if (_options.HealthCheck.CheckConfigLoaded)
        {
            var loaded = _configService.IsReady;
            checks["config"] = new { status = loaded ? "ok" : "not-ready" };
            isReady &= loaded;
        }

        if (_options.HealthCheck.CheckDatabase && _repository != null)
        {
            try
            {
                await _repository.PingAsync(ct);
                checks["database"] = new { status = "ok" };
            }
            catch (Exception ex)
            {
                checks["database"] = new { status = "fail", error = ex.Message };
                isReady = false;
            }
        }

        return (isReady, checks);
    }

    private bool IsHealthPath(string path) =>
        path.Equals(_options.HealthCheck.Path, StringComparison.OrdinalIgnoreCase) ||
        path.Equals(_options.HealthCheck.ReadyPath, StringComparison.OrdinalIgnoreCase) ||
        path.Equals(_options.HealthCheck.LivePath, StringComparison.OrdinalIgnoreCase);

    private static async Task WriteJson(HttpContext context, int status, object payload)
    {
        context.Response.StatusCode = status;
        context.Response.ContentType = "application/json";
        await context.Response.WriteAsJsonAsync(payload);
    }
}
```

### 5.5 DeploymentConfigValidator（启动验证）

```csharp
public class DeploymentConfigValidator : IHostedService
{
    private readonly IConfiguration _config;
    private readonly ILogger<DeploymentConfigValidator> _logger;

    public DeploymentConfigValidator(IConfiguration config, ILogger<DeploymentConfigValidator> logger)
    {
        _config = config;
        _logger = logger;
    }

    public Task StartAsync(CancellationToken ct)
    {
        var errors = new List<string>();
        var warnings = new List<string>();
        var kestrelSection = _config.GetSection("Kestrel:Endpoints");
        var endpoints = kestrelSection.GetChildren().ToList();
        var deployment = _config.GetSection("Gateway:Deployment").Get<DeploymentOptions>() ?? new();

        // 1. URL 格式
        foreach (var ep in endpoints)
            if (!Uri.TryCreate(ep["Url"], UriKind.Absolute, out _))
                errors.Add($"Invalid URL for endpoint '{ep.Key}': {ep["Url"]}");

        // 2. 端口冲突
        var portMap = new Dictionary<int, string>();
        foreach (var ep in endpoints)
        {
            if (!Uri.TryCreate(ep["Url"], UriKind.Absolute, out var uri)) continue;
            if (portMap.TryGetValue(uri.Port, out var existing))
                errors.Add($"Port {uri.Port} conflict: '{existing}' vs '{ep.Key}'");
            else
                portMap[uri.Port] = ep.Key;
        }

        // 3. 角色引用不存在端点
        foreach (var kvp in deployment.EndpointRoles)
            if (!endpoints.Any(e => e.Key == kvp.Key))
                errors.Add($"EndpointRoles references undefined endpoint: '{kvp.Key}'");

        // 4. Dashboard/Admin 公网绑定
        foreach (var kvp in deployment.EndpointRoles)
        {
            var ep = endpoints.FirstOrDefault(e => e.Key == kvp.Key);
            if (ep == null || !Uri.TryCreate(ep["Url"], UriKind.Absolute, out var uri)) continue;
            var isPublic = IsPublicBind(uri.Host);

            if (kvp.Value == "Dashboard" && isPublic && deployment.RequireLoopbackForDashboard)
                errors.Add($"Dashboard endpoint '{kvp.Key}' is publicly bound ({uri.Host}) but RequireLoopbackForDashboard=true");
            else if (kvp.Value == "Dashboard" && isPublic)
                warnings.Add($"SECURITY: Dashboard endpoint '{kvp.Key}' is publicly bound to {uri.Host}. Consider binding to 127.0.0.1.");

            if (kvp.Value == "Admin" && isPublic && deployment.RequireLoopbackForAdmin)
                errors.Add($"Admin endpoint '{kvp.Key}' is publicly bound ({uri.Host}) but RequireLoopbackForAdmin=true");
        }

        // 5. Mode 不一致
        if (deployment.Mode == DeploymentMode.ProxyOnly &&
            endpoints.Any(e => deployment.EndpointRoles.GetValueOrDefault(e.Key) == "Dashboard"))
            warnings.Add("ProxyOnly mode is set but Dashboard endpoint is configured. It will be ignored.");

        if (deployment.Mode == DeploymentMode.DashboardOnly &&
            endpoints.Any(e => deployment.EndpointRoles.GetValueOrDefault(e.Key) == "Proxy"))
            warnings.Add("DashboardOnly mode is set but Proxy endpoint is configured. It will be ignored.");

        if (warnings.Any())
            foreach (var w in warnings) _logger.LogWarning("⚠️ {Warning}", w);

        if (errors.Any())
        {
            foreach (var e in errors) _logger.LogError("❌ {Error}", e);
            throw new InvalidOperationException($"Deployment configuration has {errors.Count} error(s)");
        }

        _logger.LogInformation("✅ Deployment validated. Mode={Mode}, Endpoints={Count}",
            deployment.Mode, endpoints.Count);
        return Task.CompletedTask;
    }

    public Task StopAsync(CancellationToken ct) => Task.CompletedTask;

    private static bool IsPublicBind(string host) =>
        host == "0.0.0.0" || host == "*" || host == "::" || string.IsNullOrEmpty(host);
}
```

### 5.6 ConfigurationFileWatcher（热更新）

```csharp
public class ConfigurationFileWatcher : IHostedService, IAsyncDisposable
{
    private readonly IConfigurationRoot _configRoot;
    private readonly DeploymentOptions _options;
    private readonly ILogger<ConfigurationFileWatcher> _logger;
    private readonly IConfigSnapshotStore _snapshotStore;
    private readonly IGatewayAlertService _alertService;
    private readonly List<FileSystemWatcher> _watchers = new();
    private readonly string _basePath;
    private readonly string _env;
    private DateTime _lastFileWrite;
    private Timer? _debounceTimer;
    private Timer? _fallbackTimer;

    public ConfigurationFileWatcher(
        IConfiguration config,
        IOptions<DeploymentOptions> options,
        IConfigSnapshotStore snapshotStore,
        IGatewayAlertService alertService,
        ILogger<ConfigurationFileWatcher> logger)
    {
        _configRoot = config as IConfigurationRoot
            ?? throw new InvalidOperationException("IConfiguration must be IConfigurationRoot");
        _options = options.Value;
        _snapshotStore = snapshotStore;
        _alertService = alertService;
        _logger = logger;
        _basePath = AppContext.BaseDirectory;
        _env = Environment.GetEnvironmentVariable("ASPNETCORE_ENVIRONMENT") ?? "Production";
    }

    public Task StartAsync(CancellationToken ct)
    {
        if (!_options.HotReload.Enabled) return Task.CompletedTask;

        // 主：FileSystemWatcher
        foreach (var pattern in _options.HotReload.WatchedFiles)
        {
            var fileName = pattern.Replace("{Environment}", _env, StringComparison.OrdinalIgnoreCase);
            var fullPath = Path.Combine(_basePath, fileName);
            if (!File.Exists(fullPath)) continue;

            try
            {
                var watcher = new FileSystemWatcher(_basePath, fileName)
                {
                    NotifyFilter = NotifyFilters.LastWrite | NotifyFilters.Size | NotifyFilters.CreationTime
                };
                watcher.Changed += (s, e) => OnFileChanged(e.FullPath);
                watcher.Created += (s, e) => OnFileChanged(e.FullPath);
                watcher.Renamed += (s, e) => OnFileChanged(e.FullPath);
                watcher.EnableRaisingEvents = true;
                _watchers.Add(watcher);
                _logger.LogInformation("👁 Watching: {File}", fullPath);
            }
            catch (Exception ex)
            {
                _logger.LogWarning(ex, "Failed to watch {File}, will rely on polling", fullPath);
            }
        }

        // 备：兜底轮询
        if (_options.HotReload.FallbackPollSeconds > 0)
        {
            _fallbackTimer = new Timer(_ => PollingCheck(), null,
                TimeSpan.FromSeconds(_options.HotReload.FallbackPollSeconds),
                TimeSpan.FromSeconds(_options.HotReload.FallbackPollSeconds));
        }
        return Task.CompletedTask;
    }

    private void OnFileChanged(string filePath)
    {
        try { _lastFileWrite = File.GetLastWriteTimeUtc(filePath); } catch { }
        ScheduleReload(filePath, "FileChange");
    }

    private void PollingCheck()
    {
        foreach (var pattern in _options.HotReload.WatchedFiles)
        {
            var fileName = pattern.Replace("{Environment}", _env, StringComparison.OrdinalIgnoreCase);
            var fullPath = Path.Combine(_basePath, fileName);
            if (!File.Exists(fullPath)) continue;
            try
            {
                var lastWrite = File.GetLastWriteTimeUtc(fullPath);
                if (lastWrite > _lastFileWrite)
                {
                    _lastFileWrite = lastWrite;
                    ScheduleReload(fullPath, "PollingFallback");
                }
            }
            catch { }
        }
    }

    private void ScheduleReload(string filePath, string trigger)
    {
        _debounceTimer?.Dispose();
        _debounceTimer = new Timer(_ =>
        {
            try { ReloadConfig(filePath, trigger); }
            catch (Exception ex) { _logger.LogError(ex, "Reload failed"); }
        }, null, _options.HotReload.DebounceMilliseconds, Timeout.Infinite);
    }

    private void ReloadConfig(string filePath, string trigger)
    {
        _logger.LogInformation("🔄 Hot-reloading from {File} (trigger={Trigger})", filePath, trigger);

        var snapshot = CaptureSnapshot(filePath, trigger);
        _ = _snapshotStore.SaveAsync(snapshot);

        try
        {
            // ⚠️ Kestrel:Endpoints 不能热重载
            _configRoot.Reload();
            _logger.LogInformation("✅ Config hot-reloaded");
            _ = _alertService.AlertCustom("ConfigReloaded", "配置热更新",
                $"已从 {Path.GetFileName(filePath)} 重新加载", "Info");
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "❌ Reload failed, rolling back");
            _ = _alertService.AlertCustom("ConfigReloadFailed", "配置重载失败", ex.Message, "Error");
            if (_options.HotReload.RollbackOnFailure) RestoreSnapshot(snapshot);
        }
    }

    private ConfigSnapshot CaptureSnapshot(string filePath, string trigger) => new()
    {
        Timestamp = DateTime.UtcNow,
        Trigger = trigger,
        FilePath = filePath,
        Data = _configRoot.GetSection("Gateway").Get<Dictionary<string, object?>>() ?? new()
    };

    private void RestoreSnapshot(ConfigSnapshot snapshot)
    {
        try
        {
            var json = JsonSerializer.Serialize(snapshot.Data,
                new JsonSerializerOptions { WriteIndented = true });
            File.WriteAllText(snapshot.FilePath, json);
            _configRoot.Reload();
            _logger.LogInformation("↩️ Rolled back to {Timestamp}", snapshot.Timestamp);
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to rollback");
        }
    }

    public Task StopAsync(CancellationToken ct)
    {
        foreach (var w in _watchers) w.Dispose();
        _debounceTimer?.Dispose();
        _fallbackTimer?.Dispose();
        return Task.CompletedTask;
    }

    public ValueTask DisposeAsync()
    {
        foreach (var w in _watchers) w.Dispose();
        _debounceTimer?.Dispose();
        _fallbackTimer?.Dispose();
        return ValueTask.CompletedTask;
    }
}
```

```csharp
public class KestrelEndpointChangeDetector : IHostedService, IDisposable
{
    private readonly IConfiguration _config;
    private readonly EndpointRoleResolver _resolver;
    private readonly IGatewayAlertService _alertService;
    private readonly ILogger<KestrelEndpointChangeDetector> _logger;
    private readonly List<EndpointRoleMap> _originalEndpoints;
    private Timer? _checkTimer;

    public KestrelEndpointChangeDetector(
        IConfiguration config,
        EndpointRoleResolver resolver,
        IGatewayAlertService alertService,
        ILogger<KestrelEndpointChangeDetector> logger)
    {
        _config = config;
        _resolver = resolver;
        _alertService = alertService;
        _logger = logger;
        _originalEndpoints = resolver.GetAll().ToList();
    }

    public Task StartAsync(CancellationToken ct)
    {
        _checkTimer = new Timer(_ => CheckForChanges(), null,
            TimeSpan.FromSeconds(60), TimeSpan.FromSeconds(60));
        return Task.CompletedTask;
    }

    private void CheckForChanges()
    {
        var currentSection = _config.GetSection("Kestrel:Endpoints");
        var currentEndpoints = currentSection.GetChildren().ToList();

        if (currentEndpoints.Count != _originalEndpoints.Count)
        {
            NotifyChange();
            return;
        }

        var currentMap = currentEndpoints.ToDictionary(
            e => e.Key, e => e["Url"] ?? "", StringComparer.OrdinalIgnoreCase);

        if (_originalEndpoints.Any(o =>
            !currentMap.TryGetValue(o.EndpointName, out var url) ||
            !url.Contains($":{o.Port}")))
        {
            NotifyChange();
        }
    }

    private void NotifyChange()
    {
        _logger.LogWarning("⚠️ Kestrel:Endpoints changed. Restart required.");
        _ = _alertService.AlertCustom("KestrelEndpointChange", "端点配置变更",
            "Kestrel:Endpoints 修改需要重启进程才能生效", "Warning");
    }

    public Task StopAsync(CancellationToken ct) { _checkTimer?.Dispose(); return Task.CompletedTask; }
    public void Dispose() => _checkTimer?.Dispose();
}
```

```csharp
public class ConfigSnapshot
{
    public DateTime Timestamp { get; set; }
    public string Trigger { get; set; } = "";
    public string FilePath { get; set; } = "";
    public Dictionary<string, object?> Data { get; set; } = new();
}

public interface IConfigSnapshotStore
{
    Task SaveAsync(ConfigSnapshot snapshot);
    Task<ConfigSnapshot?> GetLatestAsync();
    Task<List<ConfigSnapshot>> GetHistoryAsync(int count = 5);
    Task<ConfigSnapshot?> RestoreAsync(DateTime timestamp);
}

public class FileConfigSnapshotStore : IConfigSnapshotStore
{
    private readonly string _snapshotDir;
    private readonly int _maxSnapshots;

    public FileConfigSnapshotStore(IOptions<DeploymentOptions> options)
    {
        _snapshotDir = Path.Combine(AppContext.BaseDirectory, ".config-snapshots");
        Directory.CreateDirectory(_snapshotDir);
        _maxSnapshots = options.Value.HotReload.MaxSnapshots;
    }

    public async Task SaveAsync(ConfigSnapshot snapshot)
    {
        var fileName = $"{snapshot.Timestamp:yyyyMMddHHmmssfff}-{snapshot.Trigger}.json";
        var path = Path.Combine(_snapshotDir, fileName);
        var json = JsonSerializer.Serialize(snapshot, new JsonSerializerOptions { WriteIndented = true });
        await File.WriteAllTextAsync(path, json);
        CleanupOldSnapshots();
    }

    public Task<ConfigSnapshot?> GetLatestAsync()
    {
        var files = Directory.GetFiles(_snapshotDir, "*.json")
            .OrderByDescending(f => f)
            .FirstOrDefault();
        return ReadSnapshot(files);
    }

    public Task<List<ConfigSnapshot>> GetHistoryAsync(int count = 5)
    {
        var files = Directory.GetFiles(_snapshotDir, "*.json")
            .OrderByDescending(f => f)
            .Take(count)
            .ToList();
        return Task.FromResult(files.Select(f => ReadSnapshotSync(f)).Where(s => s != null).Cast<ConfigSnapshot>().ToList());
    }

    public Task<ConfigSnapshot?> RestoreAsync(DateTime timestamp)
    {
        var file = Directory.GetFiles(_snapshotDir, "*.json")
            .FirstOrDefault(f => Path.GetFileName(f).StartsWith(timestamp.ToString("yyyyMMddHHmmssfff")));
        return ReadSnapshot(file);
    }

    private void CleanupOldSnapshots()
    {
        var files = Directory.GetFiles(_snapshotDir, "*.json")
            .OrderByDescending(f => f)
            .Skip(_maxSnapshots)
            .ToList();
        foreach (var f in files) try { File.Delete(f); } catch { }
    }

    private static async Task<ConfigSnapshot?> ReadSnapshot(string? path)
    {
        if (path == null || !File.Exists(path)) return null;
        try
        {
            var json = await File.ReadAllTextAsync(path);
            return JsonSerializer.Deserialize<ConfigSnapshot>(json);
        }
        catch { return null; }
    }

    private static ConfigSnapshot? ReadSnapshotSync(string path) => ReadSnapshot(path).GetAwaiter().GetResult();
}
```

```csharp
public static class DeploymentCli
{
    public static WebApplicationBuilder ConfigureDeploymentCli(this WebApplicationBuilder builder)
    {
        var args = Environment.GetCommandLineArgs();
        var parsed = ParseArgs(args);

        if (parsed.TryGetValue("deployment", out var mode))
            builder.Configuration["Gateway:Deployment:Mode"] = mode;
        if (parsed.TryGetValue("proxy-url", out var proxyUrl))
            builder.Configuration["Kestrel:Endpoints:Proxy:Url"] = proxyUrl;
        if (parsed.TryGetValue("dashboard-url", out var dashUrl))
            builder.Configuration["Kestrel:Endpoints:Dashboard:Url"] = dashUrl;
        if (parsed.TryGetValue("admin-url", out var adminUrl))
            builder.Configuration["Kestrel:Endpoints:Admin:Url"] = adminUrl;
        if (parsed.TryGetValue("health-url", out var healthUrl))
            builder.Configuration["Kestrel:Endpoints:Health:Url"] = healthUrl;

        return builder;
    }

    private static Dictionary<string, string> ParseArgs(string[] args)
    {
        var result = new Dictionary<string, string>(StringComparer.OrdinalIgnoreCase);
        for (int i = 0; i < args.Length; i++)
        {
            string[]? flags = args[i] switch
            {
                "--deployment"   => new[] { "deployment" },
                "--proxy-url"    => new[] { "proxy-url" },
                "--dashboard-url"=> new[] { "dashboard-url" },
                "--admin-url"    => new[] { "admin-url" },
                "--health-url"   => new[] { "health-url" },
                _ => null
            };
            if (flags != null && i + 1 < args.Length)
            {
                result[flags[0]] = args[++i];
            }
        }
        return result;
    }
}
```

## 十二、关键设计决策

### 12.1 为什么 Kestrel:Endpoints 不能热重载？

Kestrel 的 `Bind()`/`Listen()` 在 `IHostedService.StartAsync` 之前执行一次，端口绑定是**启动时一次性行为**。运行时改 `Kestrel:Endpoints`：
- ASP.NET Core 不会重新调用绑定
- `IConfigurationRoot.Reload()` 只是把新的配置值写进内存
- 端口仍使用启动时绑定的值
- 若改后端口被其他进程占用，会产生不一致状态

**解决方案**：
- 业务配置（路由/集群/策略/限流/告警）→ 可热重载
- 端点配置（端口/IP/证书）→ 检测到变化时发告警，**需要重启进程**

### 12.2 为什么需要兜底轮询？

`FileSystemWatcher` 在某些场景下会丢失事件：
- 编辑器使用 temp file + rename 写入
- 文件被其他进程锁定
- 系统 inotify 限制
- 网络文件系统的延迟

**30 秒兜底轮询**作为最后一道防线，确保配置变更不会丢失。

### 12.3 为什么 RequireLoopbackForDashboard 默认 true？

安全默认值。Dashboard 包含完整管理界面和敏感 API，误绑 `0.0.0.0` 等于公网暴露管理后台。
- 默认 true → 启动失败，强制用户意识到风险
- 显式设为 false → 允许公网访问 + 输出安全警告
- 配合 Dashboard 强认证（JWT/ApiKey）做双保险

### 12.4 为什么需要 ConfigSnapshot？

热更新可能引入新配置错误（如新策略配置有 bug）。快照机制：
- 每次重载前自动备份
- 失败时一键回滚
- 运维可手动回滚到指定版本
- 审计追溯
